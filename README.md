# BukkitFly

BukkitFlyはBukkitプラグイン作成時に、何度も書いている処理をライブラリにしたものです。

# Getting Started

```
<repositories>
	<repository>
	    <id>jitpack.io</id>
	    <url>https://jitpack.io</url>
	</repository>
</repositories>

<dependency>
    <groupId>com.github.Bun133</groupId>
    <artifactId>BukkitFly</artifactId>
    <version>Tag</version>
</dependency>

```
